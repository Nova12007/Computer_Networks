# Assignment 0 in C++

**NOTE**: This program is meant to be run on Windows Systems only.

## How to run?

To start the server, open a terminal and run the following command to build the executable -

```
g++ .\server.cpp -o http_server -lws2_32
```

Now run the following command to start the server -

```
./http_server.exe <port> <directory>
```

Replace "port" and "directory" with your respective values.
